## Options to Research

### 1. TestFlight
Apple's beta testing platform:
- Can distribute to up to 10,000 testers
- Requires Apple Developer account ($99/year)
- App goes through basic review
- Users install via TestFlight app

### 2. Progressive Web App (PWA)
Web app that acts like native:
- No app store needed
- Add to home screen
- Works offline
- Limited compared to native but very capable

### 3. Web App
Standard web application:
- Hosted online
- Accessed via browser
- Most flexible, least "app-like"

### 4. Shortcuts/Scriptable
iOS automation tools:
- Shortcuts app for simple automations
- Scriptable for JavaScript-based tools
- Limited but no developer account needed
